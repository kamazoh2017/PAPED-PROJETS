# Changelog

Tous les changements notables pour ce projet seront documentés dans ce fichier.

## [1.0.0] - 2026-03-17

### ✨ Ajouté
- Architecture Next.js 15 + React 18 + Prisma + SQLite
- Gestion complète des projets
- Système d'états pour projets (Démarrage, En cours, Terminé, Réceptionné, Clôturé)
- Gestion des entités organizationnelles
- Gestion des personnes ressources avec profils complets
- Création et attribution de tâches
- 8 statuts de tâches avec progression automatique
- Dates effectives remplies automatiquement
- Tableau Kanban par statut avec drag & drop basique
- Diagramme de Gantt avec timeline automatique
- Dashboard global avec 10+ indicateurs KPI
- Filtres avancés par projet et statut
- Calcul de vélocité par projet
- Synthèse par projet
- API RESTful complète pour toutes les entités
- Navigation multi-pages
- Interface responsive Tailwind CSS
- Validation TypeScript strict
